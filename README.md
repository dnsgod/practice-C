# Day03
- **`if`, `else if`, `else`**: 조건에 따라 실행할 코드 블록을 선택합니다.
- **`switch`**: 여러 값 중 하나에 따라 코드를 실행합니다.
- **`for`**: 정해진 횟수만큼 반복.
- **`while`**: 조건이 참인 동안 반복.
- **`do-while`**: 최소 1번 실행한 후 조건을 검토하여 반복.
